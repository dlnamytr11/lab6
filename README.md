# Ответы на вопросы:
## Что делает fetch?
### fetch() — это встроенная в JavaScript функция, которая позволяет отправлять HTTP-запросы к серверам прямо из браузера.
## Зачем нужны async/await?
### async делает функцию асинхронной, await позволяет "подождать" завершения Promise внутри async-функции и получить его результат синхронно по смыслу.
## Как работает localStorage?
### localStorage — это маленькая встроенная база данных в браузере. Она позволяет сохранять данные между перезагрузками сайта.
